This is a tool that predicts the shape of the induced emf across the phase windings of a single or multi-phase AC generator based on the machine details and the waveform of space variation of flux density provided.
Please select a file containing your flux density plot:
The accepted image file types are JPEGs and PNGs. Please make sure that the only coloured element in your image is the plot itself. The other elements of the image must be in grayscale. Examples.
The CSV file must contain either only the y values or both the x and y values. (Only the y values are needed.) The delimiter must be a comma (,). The columns may have titles. If you use MS Excel to create a CSV file, please make sure to save it as a 'CSV (Comma delimited) (*.csv)' file only.
